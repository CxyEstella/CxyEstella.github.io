## 目标

这个项目不是做成单纯的展示页，而是做一个能长期写作和查找内容的静态博客。

## 第一轮需求

- 支持 Markdown 和 MDX
- 支持图片
- 支持数学公式
- 支持彩色代码块
- 支持站内搜索
- 支持暗色模式
- 部署到 GitHub Pages

## 当前架构

内容被拆成四个集合：

```txt
blog/
reading/
notes/
projects/
```

## 决策理由

Astro 比单纯文档站更灵活，适合“笔记 + 博客 + 项目记录”的混合写法。GitHub Pages 则让部署流程足够轻，适合个人长期维护。
