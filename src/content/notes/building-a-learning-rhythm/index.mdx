## 我现在更相信节奏，而不是毅力

如果任务长期存在，真正决定能否坚持的往往不是意志力，而是系统有没有把阻力降到足够低。

## 一个最小学习循环

我把每天的学习拆成三个层次：

1. 输入：阅读、看课、看文档
2. 输出：写一小段总结
3. 固化：把结论整理成标签化笔记

### 一个简单的公式

如果把每天的有效积累记作 $G$，输入、输出、回顾的贡献可以粗略写成：

$$
G = \alpha I + \beta O + \gamma R
$$

这里真正容易被忽略的是 $R$，也就是回顾与再组织。

## 一个最小脚手架

```ts
type DailyLoop = {
  input: number;
  output: number;
  review: number;
};

export function isSustainable(loop: DailyLoop) {
  return loop.input >= 1 && loop.output >= 1 && loop.review >= 1;
}
```
